# 快贷采集端通用版本

> vue+webpack+sass+vue-router+vuex+element-ui
v1.0版本

## Build Setup

``` bash
# 安装相应的包
npm install

# 运行调试环境
npm run dev

# 打包pc版本
需要修改config/index.js文件中的build对象下的assetsPublicPath：'/';
npm run build

# 打包pad版本
需要修改config/index.js文件中的build对象下的assetsPublicPath：'';
npm run build
```
