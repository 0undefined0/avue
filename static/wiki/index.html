crud组件文档

表格属性

参数 详细解释 类型 可选值 默认值
height 表格的高度 String / Number 一 500
border 是否显示表格边框 Boolean true / false true
index 是否显示表格的序号 Boolean true / false true
selection 表格的是否可勾选 Boolean true / false true
menu 是否显示操作菜单栏 Boolean true / false true
menuWidth 操作菜单栏的宽度 String / Number 一 240
editBtn 是否显示编辑按钮 Boolean true / false true
delBtn 是否显示删除按钮 Boolean true / false true
page 是否显示分页栏目 Boolean true / false true
dic 要加载字典的字段名(调用vuex中字典接口) Array 一 一

#### 列属性

参数 详细解释 类型 可选值 默认值
label 列名称 String 一 一
prop 列对应json字段 String 一 一
width 列宽度 String / Number 一 一
fixed 是否为冻结列 Boolean true / false true
hide 隐藏列 Boolean true / false false
span 对应表单格栅列 Number 一 12
type 对应表单格类型 String select / radio / checkbox / date input
visdiplay 表单隐藏 Boolean true / false false
solt 列中支持自定义dom Boolean true / false false
overHidden 超出省略号显示 Boolean true / false false
dicData 当前列加载的字典，只有type为select / radio / checkbox 才起作用 String 一 一
rules 表单的校验例([{ required: true, message: "请输入用户名", trigger: "blur" }]) JSON 一 一

表格事件

事件名称 说明 回调参数
toggleSelection 返回当前选择的json对象 row
handleSelectionChange 返回当前选中的json数组对象 row
handleSave 表格添加成功的回调函数 row，done
handleDel 表格删除事件回调函数 row，done
handleUpdate 表格编辑事件回调函数 row，done
boxhandleClose 表单关闭前处理事件回调函数 done
boxhandleOpen 表单打开前处理事件回调函数 done
$refs.crud.toggleSelection(row); 选中那些数据传入row（当前对象） 一
$refs.crud.handleAdd(); 打开表单的增加弹窗 一

自定义列dom的例子

支持vue的solt卡槽分发添加dom内容<br />
<template slot-scope="scope" slot="username">
  <el-tag>{{scope.row.username}}</el-tag>
</template>
<template slot-scope="scope" slot="menu">
  <el-button icon="el-icon-check" size="small" @click="handleGrade(scope.row,scope.$index)">权限</el-button>
</template>
slot的名字为字段的名字，当为menu时为菜单添加dom内容<br />
